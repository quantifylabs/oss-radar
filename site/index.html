OSS.Radar FAQ

What is OSS Radar?

OSS.Radar is an AI open-source GitHub repository tracker for finding trending projects, underrated gems, and adoption risks.

How is OSS Radar different from GitHub Trending?

OSS.Radar focuses on AI categories and adds gem scores, adoption readiness, maintainer health, and abandoned-project signals.

How does the gem score work?

The gem score combines recent commits, community engagement, star acceleration, contributor diversity, documentation signals, and push recency.

What other low-rate-limit tabs could OSS.Radar add?

Good candidates are Hot This Week, Recently Active, Needs Maintainers, Docs Rich, High Velocity, and Newcomer Friendly. These reuse already-collected stars, push dates, commits, topics, and contributor counts instead of triggering extra GitHub API calls.
